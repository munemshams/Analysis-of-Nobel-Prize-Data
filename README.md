# Analysis-of-Nobel-Prize-Data
This project analyzes historical Nobel Prize winner data to uncover demographic patterns, gender representation, national trends, and notable repeat laureates. It includes:

- Full Python analysis script

- Multiple high-quality visualizations (exported automatically)

- Clean outputs suitable for academic or professional portfolios

**Key Questions Answered**

1. What is the most commonly awarded gender and birth country?

2. Which decade had the highest proportion of US-born Nobel Prize winners?

3. Which decade & category combination had the highest proportion of female laureates?

4. Who was the first woman to receive a Nobel Prize, and in what category?

5. Which individuals or organizations have won more than one Nobel Prize?

**Files Included**

- nobel_analysis.py  -  Main Python script containing:

- nobel.csv  -  Raw dataset file with all the data of the winners from 1901 to 2023.

- Exported Visualizations (.png files) - These are generated automatically when running the script.


**Visualizations Overview**

Gender Representation Over Time - Shows how male vs. female recognition has shifted during Nobel history.

Birth Country Distribution - Highlights global patterns and which countries produce the most laureates.

Female Representation by Category - Reveals which fields have historically been more or less inclusive.

US Dominance Trend - Explores how US-born winners have increased/decreased over time.
